# RHEL 8 Enterprise Infrastructure Lab
Designed and deployed an enterprise Linux infrastructure on RHEL 8 with integrated network, security, mail, web, and virtualization services to simulate a real-world enterprise environment. Configured and managed core Linux services, secure system administration, and cross-platform integration between Linux and Windows systems.

Key Implementation:
• Configured DNS server using BIND for internal domain name resolution
• Implemented DHCP server for centralized dynamic IP allocation
• Deployed Apache (HTTPD) web server with virtual host configuration
• Configured Samba file sharing for Linux–Windows integration
• Created local YUM repository for centralized and offline package management
• Implemented secure mail infrastructure using Postfix, Dovecot, and Gmail SMTP relay
• Configured KVM virtualization using QEMU, libvirt, and Virt-Manager
• Deployed and managed Windows Server 2022 virtual machine on KVM
• Applied SELinux policies, firewalld configuration, and root login security alert monitoring
• Performed system administration, troubleshooting, and service verification tasks

Technologies & Tools: RHEL 8, KVM, QEMU, Virt-Manager, BIND DNS, DHCP, Apache HTTPD, Samba, Postfix, Dovecot, SELinux, Firewalld, Linux Networking, Windows Server 2022, nmcli, Chrony
